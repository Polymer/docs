Get Started

Ready to dive in? You're moments away from building your first element or your first app.

From here, you can also learn more about Polymer and the App Toolbox, watch some Polycasts, and set yourself up with the Polymer CLI.

Use Elements

The easiest way to dip your toe in the Web Components pool is to grab some elements and try them out.

The Polymer Element Catalog houses several collections of Web Components that you can use in your apps.

The elements in the catalog have been built with Polymer, but – since Web Components are inherently interoperable – can be used on any site.

Browse elements Learn more

Build Elements

If you're ready to create your own elements, the Polymer library has you covered.

Custom elements are ideal for extending HTML semantics, branding and polishing your UI, or embedding rich functionality in your site.

Build elements for your own use, to share with your organization or to save the world, one widget at a time.

Build an element Learn more

Build Apps

You can also use Web Components to build apps from the ground up.

The Polymer App Toolbox helps you build Progressive Web Apps using Web Components and a set of complementary new web platform features.

Use the pieces of the Toolbox individually, or use them together to deliver truly modern apps that launch quickly, respond instantly and work from anywhere.

Build an app Learn more

Get the tools


npm install -g polymer-cli
mkdir my-app
cd my-app
polymer init

Polymer CLI is a command-line interface for building elements with Polymer and applications with the Polymer App Toolbox. It streamlines your workflow, making it easy to manage projects ranging from simple components to complex web apps.
